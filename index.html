Steering N'Wheel for Nokia N900.

This is a Hobby Project! I'm worked on a Project that contains: Playing racing games with a Smartphone! It gives you the possibility to controll the car, completly from smartphone. Accelerate, Brake, Shift up/down, handbrake and more...

Programming

It was completly programmed with Python, it's also Cross Platform

Testing

I tested it with different racing games! Live for Speed, Vdrift and torcs. But you can use it, with all racing games that gives you the possibility, to control it with the mouse.

Principle and operation of the program

It runs over Client/Server. You install the client on your Phone (Nokia N900) and the server on the PC. Start the Server from PC. If the Game begins, connect to the server and it runs.

Future
